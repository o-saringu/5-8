mousePressed変数

サンプルコードの結果

サンプルコード

　　 void setup(){

　　 　size(240, 120);

　　 　strokeWeight(30);　　 　

　　 }

　　

　　 void draw(){

　　 　background(204);

　　 　stroke(102);

　　 　line(40, 0, 70, height);

　　 　if(mousePressed==true){

　　 　　stroke(0);

　　 　　}

　　 　line(0, 70, width, 50);

　　 }

説明文

　mousePressed変数が取り得る値は（true）か（false）の2種類である。

　マウスのボタンがクリックされるとmousePressed変数は真（true）の値を得る。

　マウスのボタンがクリックされていないと偽（false）の値を得ている。

　mousePressed変数とif構文を組み合わせることでコードを実行すべきかどうか判別することができる。

　例では、マウスのボタンがクリックされている間はmousePressedの値が（true）となりstroke(0)が実行される仕組みとなっている。

　mousePressed変数はそれ自体が真か偽を示すため、例では使用しているが下記のように示し、必ず演算子を使用し明確な条件を与える必要はない。
